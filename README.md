## Описание программы


**Training Planner** - это GUI-приложение для управления списком тренировок. Приложение позволяет добавлять тренировки, фильтровать их по типу и длительности, сохранять и загружать данные в формате JSON, а также удалять отдельные записи или весь список целиком.


### Возможности:
- ✅ Добавление тренировок
- ✅ Фильтрация тренировок по типу 
- ✅ Сохранение данных в JSON-файл
- ✅ Загрузка данных из JSON-файла
- ✅ Очистка всего списка 
- ✅ Валидация вводимых данных


## Инструкция по запуску
- python main.py

### Требования:
- Python 3.7 или выше
- Установленная библиотека Tkinter (обычно идёт в комплекте с Python)
- Установленная библиотека json


### Запуск:
1. Откройте терминал или командную строку
2. Перейдите в директорию проекта:
   ```bash
   cd myProject
   ```
3. Запустите приложение:
   ```bash
   python main.py
   ```


## Примеры использования


### Добавление тренировки:
1. Заполните все поля формы:
2. Нажмите кнопку **"Добавить тренировку"**
3. Фильм появится в списке внизу окна


### Сохранение и загрузка:
- Нажмите **"Сохранить в JSON"** - данные сохранятся в файл `training.json`




## Валидация данных


Приложение проверяет:
- Все поля должны быть заполнены
